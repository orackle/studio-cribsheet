EQ Ranges

Quick bands + common moves

  • Sub 20–60 — feel
  • Bass 60–250 — weight / mud
  • Low-mids 250–500 — boxy
  • Mids 500–2k — body / presence
  • High-mids 2–6k — clarity / harsh
  • Highs 6–12k — air

Compressor Starters

Attack/Release by material

  • Vocals 3–5:1 • medium A/R
  • Bass 4–8:1 • faster R
  • Drum bus 2–4:1 • slow A, fast R
  • Parallel add 10–30% squashed

Loudness Targets

Streaming context & headroom

  • Streaming: ~ −14 LUFS
  • True peak: ≤ −1 dBTP
  • Measure with BS.1770 / EBU R128 meters
  • Don’t chase exact LUFS; protect dynamics

Vocal Chain

Fast, clean starting recipe

  • HPF: male 70–100 Hz • female 90–120 Hz
  • Cut: tame 200–400 Hz if boxy
  • Compress: ~3–5 dB GR • medium A/R
  • De-ess: 5–8 kHz as needed
  • Air: gentle shelf 8–12 kHz

Export Checklist

One pass before you bounce

  • Format: WAV 24-bit • 44.1/48 kHz
  • TP ceiling: ≈ −1 dBTP • check inter-sample
  • Silence: tidy heads/tails
  • Metadata: ISRC/artist/title embedded
